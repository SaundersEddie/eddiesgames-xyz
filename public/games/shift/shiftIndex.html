    How to Play

    Goal: Slide tiles and merge matching numbers.

    Use arrow keys, WASD, or swipe on mobile to move every tile on the board at once.

    When two matching numbers collide, they merge into a higher value.

    Keep merging, build bigger numbers, and push your score as high as possible before the board locks up.

    Daily mode uses the same seeded run for everyone. Random mode gives you a fresh run every time.
