How to Play

Goal: Fill every row, column, and 3×3 box with the numbers 1–9.

Each number can appear only once in each row, column, and 3×3 box.

Click a cell, then use the number buttons or keyboard to enter a number. Use Notes mode for pencil marks.

Wrong final entries cost a life. Run out of lives and the puzzle is over.

Daily mode uses the same puzzle for everyone. Easy, Medium, and Hard are free-play games and do not save scores.
